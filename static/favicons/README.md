# Favicons

Place your favicon file directly in the `static/` directory.

## Required File

**favicon.ico** - Main favicon
- Place at: `static/favicon.ico`
- Recommended: 16x16, 32x32, or 48x48 pixels
- This is the standard favicon format that all browsers support

## How to Add Your Favicon

1. Create or obtain your favicon image (square image, ideally 32x32 or 64x64 pixels)
2. Convert it to `.ico` format (or use a PNG and rename it)
3. Place the file as `static/favicon.ico`
4. Rebuild your site with `hugo`

The favicon will automatically appear in browser tabs once the file is in place.

## Generating Favicons

You can use online tools to generate favicons from your logo/image:
- [Favicon.io](https://favicon.io/) - Simple and free
- [RealFaviconGenerator](https://realfavicongenerator.net/) - Comprehensive, generates all formats
- [Favicon Generator](https://www.favicon-generator.org/) - Another option

These tools can generate `.ico` files from PNG/JPG images.
